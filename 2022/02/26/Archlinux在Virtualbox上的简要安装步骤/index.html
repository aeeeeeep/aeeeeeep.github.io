Archlinux在Virtualbox上的简要安装步骤

准备工作

  • 从清华源下载最新Archlinux官网镜像
  • 下载安装对应当前系统的最新VirtualBox安装程序

安装系统

打开Virtualbox

检查网络，正常；
设置时间

查看硬盘分区

擦除分区表sgdisk -Z /dev/sda

创建大小为 260M 的 ESP 分区sgdisk -n=0:0:+260M -t=0:ef00 /dev/sda

使用剩余空间创建系统分区：sgdisk -N=0 /dev/sda

格式化 ESP 分区为 FAT32：mkfs.fat -F 32 /dev/sda1

格式化系统分区为 xfs：mkfs.xfs /dev/sda2

挂载系统分区：mount /dev/sda2 /mnt

设置中国镜像源


删除#Server = 前面的#以启用镜像

设置本地化环境：export LANG=C

安装所需软件包到系统分区：`pacstrap /mnt base linux linux-firmware bash-completion xfsprogs vim networkmanager pacman-contrib sudo amd-ucode yay git neofetch

成功安装完成后生成 fstab 文件：genfstab -U /mnt > /mnt/etc/fstab

进入新系统环境：arch-chroot /mnt

设置时区为中国：ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

同步硬件时钟：hwclock --systohc

设置系统主机名：echo ${hostname} > /etc/hostname

编辑 hosts 文件vim /etc/hosts并添加以下行：

编辑本地化配置文件vim /etc/locale.gen
删除以下行前面的#

设置默认本地化环境：echo LANG=en_US.UTF-8 > /etc/locale.conf

执行SUDO_EDITOR=nano visudo编辑 sudo 配置文件，删除


前面的#让 wheel 用户组可以使用sudo获取 root 权限

创建一个新用户并加入 wheel 用户组：

设置新用户密码：passwd pacaep

安装引导管理程序 GRUB：pacman -S grub

安装 GRUB 至 ESP 分区：grub-install --removable --target=x86_64-efi --efi-directory=/boot

生成 GRUB 配置文件：grub-mkconfig -o /boot/grub/grub.cfg

启用网络管理器：sudo systemctl enable --now NetworkManager

启用从网络同步系统时间：sudo timedatectl set-ntp true

启用本地 DNS 解析器：sudo systemctl enable --now systemd-resolved

创建 DNS 解析文件软链接：sudo ln -sf /run/systemd/resolve/stub-resolv.conf /etc/resolv.conf

启用每周自动清理不需要的 Pacman 软件包缓存：sudo systemctl enable paccache.timer

安装KDE桌面环境、显示管理器、终端（Konsole）以及 Noto 字体集：sudo pacman -S --noconfirm plasma sddm konsole noto-fonts noto-fonts-cjk noto-fonts-emoji

安装 Plasma 可选依赖：sudo pacman -S --asdeps gst-plugins-good packagekit-qt5 xsettingsd

打开系统设置，按需求设置

安装完成
