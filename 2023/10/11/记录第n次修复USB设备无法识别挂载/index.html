记录第n次修复USB设备无法识别挂载

2023-10-11
archlinux

摘要

archlinux 滚着滚着就不能识别 USB 设备了，记录一下修复过程

准备

一个U盘

过程

使用 lsusb 可以识别到

sudo dmesg | grep USB 显示如下，可以识别到

lsblk 显示如下，没有USB设备

sudo fdisk -l 显示如下，没有USB设备

df -h 显示如下，没有usb设备

总结 lsusb，sudo dmesg | grep USB 可以识别，lsblk，fdisk -l，df -h 不能识别到，推测是因为系统没有自动挂载

解决

安装 udisks2 和 udevil

启用devmon服务

reboot 后即可生效
